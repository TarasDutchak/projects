Backend Developer

About this job

  • Job type: Full-time
  • Experience level: Mid-Level
  • Role: Backend Developer
  • Location: Kyiv, New York

Technologies

  • Python
  • Git
  • MySQL
  • HTML

Job description

We’re expanding our development team and looking for a talented backend engineer who is proficient in Python. If you are passionate about technology, constantly seeking to learn and improve skill set, then you are the type of person we are looking for.

Requirements:

  • At least 4 years of experience in software development.
  • 2+ years working with Python 3 and Django platform.
  • Experience working with relational (MySQL, PostgreSQL) and non-relational databases (Redis, MongoDB).
  • Strong knowledge of the common web standards and data formats (HTML, CSS, JSON, etc.).
  • Familiar with the concepts behind modern web apps (REST, AJAX).
  • Experience with Docker and Git.
  • Practical experience of troubleshooting and performance optimization.
  • Experience working in a Linux environment.
  • An intermediate or higher level of written and spoken English.
  • Ability to work individually or as part of a team.

As advantage:

  • Experience with Agile, TDD and Continuous Integration.
  • Experience working on low latency real-time application is a big plus.
  • Experience with Python frameworks: Flask, Django Rest Framework, SQLAlchemy.
  • Experience with Python libraries: Asyncio, ZMQ.
  • Familiar with Javascript and Javascript frameworks.
  • Refactoring experience will be a big plus.

As a Python developer, you will:

  • Design, develop, debug, test and document new services, APIs for internal use.
  • Maintain existing services, APIs and resolve problem reports.
  • Integrate 3rd party services, APIs.
  • Improve internal and external APIs.
